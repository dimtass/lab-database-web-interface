Add a new part:


To add a new part in the database you need to add at least
the 'Part name'. Additionally, you can add a description,
which is also shown in the items list and add additional
tags in the 'Tags' field, which are used from the search
function in the main page. Use the 'Choose File' buttons
to upload an image and the datasheet of the part.

If you're also using the esp8266 firmware for indicating
the part location, then add the led index of the WS2812B
stripe.
